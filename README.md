# TypeMaster
To run the project on your system follow these steps:-
1) Create a New Folder on your system named "typing-test".
2) Copy the contents of this repository (public folder and server.js) on your system.
3) Now open terminal and change the location to "......\typing-test" .
4) Now first run the commands "npm init --y" and then "npm i express" on your terminal.
5) Now check if "package.json" is created or not and inside this file change type: "commonjs" to type:"module".
6) Finally go back to terminal and run command - "node server.js"
